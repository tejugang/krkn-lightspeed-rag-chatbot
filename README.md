# krkn-lightspeed-rag-chatbot
This is a test chatbot that uses the pod scenarios documentation to generate responses for users using the RAG model. 

Download this model and add it to the models folder on your local machine: https://huggingface.co/TheBloke/Llama-2-7B-Chat-GGUF/blob/main/llama-2-7b-chat.Q4_K_M.gguf

Instructions on running the chatbot: 

To run in the terminal: type `python3 main.py` in the terminal inside the project folder
